# JSON Formatter

Format, validate, and minify JSON with syntax highlighting.

## Features

- **Format**: Pretty-print JSON with 2-space indentation
- **Minify**: Compress JSON to single line
- **Syntax Highlighting**: Color-coded keys, strings, numbers, booleans, and null values
- **Validation**: Real-time JSON validation with error messages
- **Copy Output**: One-click copy to clipboard

## Usage

1. Paste or type JSON in the input panel
2. Output automatically formats with syntax highlighting
3. Use toolbar buttons to minify or copy

## Privacy

All processing happens locally in your browser. No data is sent to any server.
