# Password Generator

Generate secure random passwords with customizable options.

## Features

- **Cryptographically Secure**: Uses Web Crypto API for true randomness
- **Customizable Length**: 4 to 64 characters
- **Character Options**: Uppercase, lowercase, numbers, symbols
- **Strength Indicator**: Visual password strength meter
- **One-Click Copy**: Copy to clipboard instantly

## Usage

1. Adjust length slider
2. Toggle character sets
3. Click Generate for a new password
4. Copy to clipboard

## Privacy

All generation happens locally. No passwords are ever transmitted or stored.
