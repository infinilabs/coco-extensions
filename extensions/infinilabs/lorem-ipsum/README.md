# Lorem Ipsum Generator

Generate placeholder text for design and development.

## Features

- **Paragraphs**: Generate full paragraphs
- **Sentences**: Generate individual sentences
- **Words**: Generate word lists
- **Classic start**: Always begins with "Lorem ipsum dolor sit amet..."
- **Copy**: One-click copy to clipboard

## Usage

1. Select output type (paragraphs, sentences, or words)
2. Enter desired count
3. Click Generate
4. Copy the result

## About Lorem Ipsum

Lorem Ipsum has been the industry's standard dummy text since the 1500s. It's used in:

- Web design mockups
- Print layouts
- Typography samples
- UI prototypes

## Privacy

All generation happens locally. No data is sent anywhere.
