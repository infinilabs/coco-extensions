# Markdown Preview

Live preview of Markdown text.

## Features

- **Real-time rendering** as you type
- **Side-by-side view** - edit and preview together
- **Dark mode support**

## Supported Syntax

- Headers (h1-h6)
- **Bold** and *italic* text
- ~~Strikethrough~~
- `Inline code`
- Code blocks with syntax highlighting
- [Links](url)
- ![Images](url)
- Blockquotes
- Ordered and unordered lists
- Horizontal rules

## Usage

1. Type or paste Markdown in the left pane
2. See rendered preview in the right pane
3. Copy from preview as needed

## Privacy

All rendering happens locally. No data is sent anywhere.
