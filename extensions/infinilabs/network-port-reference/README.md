# Network Port Reference

Searchable reference of common network ports and services.

## Features

- **40+ common ports** - Web, mail, database, remote access
- **Search** - By port number, name, or description
- **Filter** - All, TCP, UDP, Web, Mail, Database
- **Click to copy** - Copy port number instantly

## Port Categories

| Category | Examples |
|----------|----------|
| Web | HTTP (80), HTTPS (443), 8080, 8443 |
| Mail | SMTP (25/587), POP3 (110), IMAP (143) |
| Database | MySQL (3306), PostgreSQL (5432), MongoDB (27017) |
| Remote | SSH (22), RDP (3389), VNC (5900) |

## Usage

1. Search for a port or service
2. Use filter tabs to narrow results
3. Click any port to copy the number

## Privacy

All data is local. No network requests.
