# Data URL Generator

Create data URLs from text content or files.

## Features

- **Text mode** - Convert text with custom MIME type
- **File mode** - Drag & drop or browse for files
- **Image preview** - Shows preview for image files
- **Size info** - Original vs encoded size

## Supported MIME Types (Text)

- text/plain
- text/html
- text/css
- text/javascript
- application/json
- application/xml
- image/svg+xml

## Usage

### Text Mode
1. Select MIME type from dropdown
2. Enter or paste text content
3. Data URL is generated automatically

### File Mode
1. Switch to "File" tab
2. Drag & drop or click to browse
3. Data URL is generated from file

## Use Cases

- Embedding small images in CSS
- Creating bookmarklets
- Inline SVG in HTML
- Self-contained HTML documents

## Privacy

All processing happens locally. Files are never uploaded.
