# HTTP Status Codes

Searchable reference of HTTP status codes.

## Categories

| Range | Category | Color |
|-------|----------|-------|
| 1xx | Informational | Blue |
| 2xx | Success | Green |
| 3xx | Redirection | Yellow |
| 4xx | Client Error | Red |
| 5xx | Server Error | Dark Red |

## Features

- **50+ status codes** - All standard HTTP status codes
- **Search** - By code number, name, or description
- **Filter by category** - Click tabs to filter by range
- **Color coded** - Visual indication of status type

## Common Codes

| Code | Name | Use |
|------|------|-----|
| 200 | OK | Success |
| 201 | Created | Resource created |
| 301 | Moved Permanently | Permanent redirect |
| 400 | Bad Request | Client error |
| 401 | Unauthorized | Auth required |
| 403 | Forbidden | Access denied |
| 404 | Not Found | Resource missing |
| 500 | Internal Server Error | Server error |

## Privacy

All data is local. No network requests.
