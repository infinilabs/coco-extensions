# JSON Path Finder

Find paths to values in JSON data.

## Features

- **Interactive tree view** - Expand and explore JSON structure
- **Click to copy path** - Click any value to see its path
- **Multiple path formats**:
  - Dot notation: `user.name`
  - Bracket notation: `user["name"]`
  - Array indices: `roles[0]`
- **Syntax highlighting** by type

## Usage

1. Paste JSON in the left pane
2. Click any key or value in the tree
3. Path appears in the top bar
4. Click Copy to copy the path

## Use Cases

- Finding paths for jq queries
- Debugging API responses
- Writing JSONPath expressions
- Understanding nested data structures

## Privacy

All processing happens locally. No data is sent anywhere.
