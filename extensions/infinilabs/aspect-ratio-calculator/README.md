# Aspect Ratio Calculator

Calculate aspect ratios and resize dimensions.

## Features

- **Ratio calculation** - Get simplified ratio from any dimensions
- **Decimal display** - Shows ratio as decimal (e.g., 1.778)
- **Common presets** - 16:9, 4:3, 1:1, 21:9, 9:16, 3:2, 2:3, 5:4
- **Resize calculator** - Maintain aspect ratio when resizing
- **Lock/unlock** - Toggle ratio lock for free resize

## Usage

1. Enter original width and height
2. View calculated aspect ratio
3. Use presets or enter new dimensions
4. Lock maintains ratio when resizing

## Common Ratios

| Ratio | Use Case |
|-------|----------|
| 16:9 | HD/4K video, modern displays |
| 4:3 | Classic TV, iPad |
| 1:1 | Instagram, profile pictures |
| 9:16 | Mobile video, TikTok, Stories |
| 21:9 | Ultrawide, cinema |
| 3:2 | DSLR photos |

## Privacy

All calculations happen locally. No data is sent anywhere.
