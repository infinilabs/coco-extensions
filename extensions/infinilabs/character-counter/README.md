# Character Counter

Detailed real-time text analysis.

## Statistics

### Primary
- **Characters** - Total character count
- **Words** - Word count
- **Lines** - Line count

### Detailed
- **Characters (no spaces)** - Excluding whitespace
- **Sentences** - Estimated sentence count
- **Paragraphs** - Paragraph count
- **Bytes (UTF-8)** - Size in bytes
- **Avg. Word Length** - Average characters per word
- **Reading Time** - Estimated at 200 WPM

## Usage

1. Type or paste text
2. Statistics update in real-time

## Use Cases

- Check text length limits (tweets, SMS)
- Estimate document size
- Analyze writing complexity
- Track reading time

## Privacy

All analysis happens locally. No data is sent anywhere.
