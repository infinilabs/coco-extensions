# Base Converter

Convert numbers between different number bases.

## Features

- **Decimal** (Base 10) - Standard numbers
- **Hexadecimal** (Base 16) - Used in programming, colors
- **Octal** (Base 8) - Unix file permissions
- **Binary** (Base 2) - Machine code

## Usage

1. Enter a number in any field
2. Other fields update automatically
3. Click Copy to copy a value

## Examples

| Decimal | Hex | Octal | Binary |
|---------|-----|-------|--------|
| 10 | A | 12 | 1010 |
| 255 | FF | 377 | 11111111 |
| 1024 | 400 | 2000 | 10000000000 |

## Privacy

All conversions happen locally. No data is sent anywhere.
