# Binary Text Converter

Convert between text and binary/octal/decimal/hex representation.

## Features

- **Multiple encodings** - Binary, Octal, Decimal, Hex
- **Configurable separator** - Space, none, dash, newline
- **Bidirectional** - Encode text or decode numbers
- **Swap direction** - Click swap button to reverse

## Encodings

| Encoding | Base | Example ('A') |
|----------|------|---------------|
| Binary | 2 | 01000001 |
| Octal | 8 | 101 |
| Decimal | 10 | 65 |
| Hex | 16 | 41 |

## Usage

### Encode Text
1. Type text in the top field
2. Binary/code appears below
3. Select encoding format

### Decode
1. Click the swap button (⇅)
2. Enter binary/code in the bottom field
3. Text appears above

## Examples

**"Hello" in Binary:**
```
01001000 01100101 01101100 01101100 01101111
```

**"Hello" in Hex:**
```
48 65 6C 6C 6F
```

## Privacy

All processing happens locally.
