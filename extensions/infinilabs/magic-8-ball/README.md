# Magic 8-Ball

Ask the Magic 8-Ball a yes/no question and receive a mystical answer.

## Features

- Classic Magic 8-Ball with 20 traditional answers
- 10 positive, 5 neutral, and 5 negative responses
- Shake animation when revealing answers
- Color-coded answers (green/yellow/red)
- Cryptographically random answer selection

## Usage

1. Type your yes/no question (optional)
2. Click the ball or press Enter
3. Watch the shake animation
4. Receive your mystical answer

## Privacy

All answers are generated locally using Web Crypto API. No data is sent anywhere.
