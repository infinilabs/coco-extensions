# CSS Unit Converter

Convert between CSS units with configurable base values.

## Supported Units

| Unit | Description |
|------|-------------|
| px | Pixels |
| rem | Root em (relative to root font size) |
| em | Em (relative to parent font size) |
| pt | Points (1pt = 1.333px) |
| vw | Viewport width percentage |
| vh | Viewport height percentage |
| % | Percentage of root font size |

## Features

- **Configurable base values** - Root font size, viewport dimensions
- **Quick presets** - Common size buttons
- **Click to copy** - Copy any converted value
- **Real-time conversion** - Updates as you type

## Default Settings

- Root Font Size: 16px
- Viewport Width: 1920px
- Viewport Height: 1080px

## Common Conversions (16px root)

| px | rem |
|----|-----|
| 8px | 0.5rem |
| 12px | 0.75rem |
| 14px | 0.875rem |
| 16px | 1rem |
| 24px | 1.5rem |
| 32px | 2rem |

## Usage

1. Enter a value and select the input unit
2. View conversions to all other units
3. Click any result to copy

## Privacy

All calculations happen locally.
