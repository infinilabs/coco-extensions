# Slug Generator

Convert text to URL-friendly slugs.

## Features

- **Real-time conversion** as you type
- **Separator options**: Hyphen, underscore, or dot
- **Lowercase toggle**
- **Max length limit** with smart word boundary detection
- **Accent handling** - converts àéïöü to aeiou

## Usage

1. Enter text (title, name, etc.)
2. Adjust options as needed
3. Copy the generated slug

## Examples

| Input | Slug |
|-------|------|
| Hello World! | hello-world |
| My Blog Post (2024) | my-blog-post-2024 |
| Café & Résumé | cafe-resume |
| 10 Best Tips | 10-best-tips |

## Use Cases

- Create URL permalinks
- Generate file names
- Create SEO-friendly URLs
- Name database identifiers

## Privacy

All processing happens locally. No data is sent anywhere.
