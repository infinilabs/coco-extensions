# Stopwatch

A simple stopwatch with lap times.

## Features

- **Start/Pause/Resume**: Control the stopwatch
- **Lap Times**: Record lap times with difference from previous lap
- **Copy All**: Copy all lap times to clipboard
- **Reset**: Clear everything and start fresh

## Usage

1. Click Start to begin timing
2. Click Lap to record lap times
3. Click Pause to pause, Resume to continue
4. Click Reset to clear
