# Chmod Calculator

Calculate Unix file permissions in numeric and symbolic notation.

## Features

- **Visual grid** - Check boxes for read, write, execute
- **Three outputs** - Numeric (755), symbolic (rwxr-xr-x), command
- **Common presets** - 755, 644, 600, 777, 700, 666
- **Copy buttons** - Copy any format instantly

## Permission Values

| Permission | Numeric | Symbolic |
|------------|---------|----------|
| Read | 4 | r |
| Write | 2 | w |
| Execute | 1 | x |

## Common Permissions

| Value | Use Case |
|-------|----------|
| 755 | Executables, directories |
| 644 | Regular files |
| 600 | Private files |
| 777 | Full access (use carefully) |
| 700 | Owner-only directory |

## Usage

1. Check the permissions you need
2. View numeric, symbolic, and command output
3. Click Copy to copy any format

## Privacy

All calculations happen locally. No data is sent anywhere.
