# Text Sorter

Sort lines of text with various options.

## Sort Options

- **Alphabetical (A-Z / Z-A)** - Standard alphabetical sort
- **Numeric (0-9 / 9-0)** - Sort by number value
- **By Length** - Sort by line length
- **Random Shuffle** - Randomize line order

## Additional Options

- **Remove duplicates** - Keep only unique lines
- **Trim lines** - Remove leading/trailing whitespace
- **Ignore case** - Case-insensitive sorting

## Usage

1. Paste lines of text
2. Select sort type and options
3. Click Sort
4. Copy the result

## Use Cases

- Sort lists alphabetically
- Remove duplicate entries
- Randomize quiz questions
- Organize configuration lines

## Privacy

All processing happens locally. No data is sent anywhere.
