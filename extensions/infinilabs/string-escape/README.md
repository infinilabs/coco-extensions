# String Escape

Escape and unescape strings for various formats.

## Supported Formats

- **JSON** - Escape/unescape for JSON strings
- **XML** - XML entity encoding (&amp; &lt; &gt; etc.)
- **HTML** - HTML entity encoding
- **URL** - URL encoding (percent encoding)
- **Regex** - Escape special regex characters

## Usage

1. Select the format tab
2. Enter text in the input field
3. Click Escape or Unescape
4. Copy the result

## Examples

| Format | Input | Escaped |
|--------|-------|---------|
| JSON | Hello "World" | Hello \\"World\\" |
| XML | <tag> | &lt;tag&gt; |
| URL | hello world | hello%20world |
| Regex | a.b*c | a\\.b\\*c |

## Privacy

All processing happens locally. No data is sent anywhere.
