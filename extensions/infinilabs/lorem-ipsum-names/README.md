# Lorem Ipsum Names

Generate random fake names for testing and mockups.

## Features

- **Configurable count** - Generate 1-100 names
- **Gender filter** - Any, male, or female
- **Optional email** - Auto-generated email addresses
- **Click to copy** - Copy individual names
- **Copy all** - Copy entire list at once

## Use Cases

- UI mockups and wireframes
- Test data generation
- Database seeding
- Placeholder content

## Options

| Option | Description |
|--------|-------------|
| Count | Number of names to generate (1-100) |
| Gender | Filter by male, female, or any |
| Email | Include fake email addresses |

## Output Format

Click a name to copy:
```
John Smith
john.smith@email.com
```

Copy all (CSV-like):
```
John Smith, john.smith@email.com
Jane Doe, jane.doe@mail.com
```

## Privacy

All names are generated locally using randomized data. No network requests.
