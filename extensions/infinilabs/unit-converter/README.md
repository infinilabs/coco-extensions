# Unit Converter

Convert between common units of measurement.

## Categories

- **Length**: Meters, Kilometers, Miles, Feet, Inches, etc.
- **Weight**: Kilograms, Grams, Pounds, Ounces, etc.
- **Temperature**: Celsius, Fahrenheit, Kelvin
- **Volume**: Liters, Gallons, Cups, Fluid Ounces, etc.
- **Area**: Square Meters, Acres, Hectares, etc.
- **Speed**: m/s, km/h, mph, Knots, etc.

## Features

- Real-time conversion as you type
- Swap button to reverse conversion direction
- Smart number formatting for very large/small values

## Privacy

All calculations happen locally. No data is sent anywhere.
