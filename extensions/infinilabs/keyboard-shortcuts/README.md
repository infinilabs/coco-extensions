# Keyboard Shortcuts

Reference of common keyboard shortcuts for popular applications.

## Applications

- **General** - System-wide shortcuts
- **Browser** - Chrome, Firefox, Safari
- **VS Code** - Code editor
- **Terminal** - Command line
- **Finder** - File manager (macOS/Windows Explorer)

## Features

- **Platform toggle** - macOS vs Windows shortcuts
- **Search** - Filter by action name
- **Category tabs** - Browse by application
- **Visual keys** - Styled key caps

## Common Shortcuts

| Action | macOS | Windows |
|--------|-------|---------|
| Copy | ⌘C | Ctrl+C |
| Paste | ⌘V | Ctrl+V |
| Undo | ⌘Z | Ctrl+Z |
| Find | ⌘F | Ctrl+F |
| Save | ⌘S | Ctrl+S |

## Usage

1. Select your platform (auto-detected)
2. Choose application category
3. Search for specific shortcuts

## Privacy

All data is local. No network requests.
