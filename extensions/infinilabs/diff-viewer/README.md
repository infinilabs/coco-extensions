# Diff Viewer

Compare two text blocks and see the differences highlighted.

## Features

- **Line-by-line comparison** using LCS (Longest Common Subsequence) algorithm
- **Color-coded output**:
  - 🟢 Green: Added lines
  - 🔴 Red: Deleted lines
  - Gray: Unchanged lines
- **Statistics**: Shows count of added/deleted lines
- **Swap**: Exchange original and modified text

## Usage

1. Paste original text in the left field
2. Paste modified text in the right field
3. Click Compare
4. View highlighted differences

## Use Cases

- Compare code versions
- Check config file changes
- Review text edits
- Verify data modifications

## Privacy

All comparisons happen locally. No data is sent anywhere.
