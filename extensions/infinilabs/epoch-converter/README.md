# Epoch Converter

Convert between Unix timestamps and human-readable dates.

## Features

- **Live Current Time**: Shows current Unix timestamp, updates every second
- **Timestamp → Date**: Convert Unix timestamps to local date/time
- **Date → Timestamp**: Convert dates to Unix timestamps
- **Seconds/Milliseconds**: Toggle between second and millisecond precision
- **Click to Copy**: Click any result to copy to clipboard

## Usage

### Timestamp to Date
1. Select seconds or milliseconds
2. Enter a Unix timestamp
3. See the converted date

### Date to Timestamp
1. Select a date/time
2. See both seconds and milliseconds formats

## Common Timestamps

- `0` = January 1, 1970 (Unix Epoch)
- `1000000000` = September 9, 2001
- `1500000000` = July 14, 2017
- `2000000000` = May 18, 2033

## Privacy

All conversions happen locally. No data is sent anywhere.
