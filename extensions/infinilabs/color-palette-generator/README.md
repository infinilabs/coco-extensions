# Color Palette Generator

Generate harmonious color palettes from a base color.

## Features

- **Color picker** - Select base color visually
- **Hex input** - Enter color code directly
- **Harmony types** - Multiple color harmony algorithms
- **Shades** - Auto-generated light/dark shades
- **Click to copy** - Copy any color code

## Harmony Types

| Type | Description |
|------|-------------|
| Complementary | Base + opposite on color wheel |
| Analogous | Base + adjacent colors (±30°) |
| Triadic | Three evenly spaced colors (120°) |
| Tetradic | Four colors (90° apart) |
| Split Complementary | Base + two adjacent to complement |
| Monochromatic | Single hue with varying lightness |

## Usage

1. Select a base color using the picker or hex input
2. Choose a harmony type
3. Click any swatch to copy its hex code
4. Use "Copy All" to copy entire palette

## Color Theory

- **Complementary** - High contrast, vibrant
- **Analogous** - Harmonious, comfortable
- **Triadic** - Vibrant, balanced
- **Split Complementary** - Balanced contrast
- **Monochromatic** - Cohesive, elegant
